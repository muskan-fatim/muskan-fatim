# DOM Assignment: Random Joke Button

In this assignment, we learned how the Document Object Model (DOM) makes our work easier by allowing us to manipulate web page content dynamically. Using DOM methods, we added functionality to a button that displays a random joke every time it is clicked.

## Features

- **Random Joke Button**: Click the button to see a new random joke each time.
- **Dynamic Content Update**: The joke displayed on the page updates dynamically without refreshing the page.

## Technologies Used

- **HTML**: For the structure of the web page.
- **CSS**: For styling the button and page layout.
- **JavaScript**: For DOM manipulation and handling button click events.

## How to Use

1. Clone the repository to your local machine.
2. Open the `index.html` file in your browser.
3. Click the "Tell me a joke" button to see a new joke.

## Code Overview

In the JavaScript file, we used the following key functions:

document.getElementById(): To select the button and joke display elements.
addEventListener('click'): To add the click event listener to the button.

## Example

Clicking the button shows a random joke like this:

> Why don't scientists trust atoms? Because they make up everything
